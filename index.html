Android Messenger

Developed an Android based messenger with the Android SDK. Messenger allowed for two or more AVDs (Android Virtual Devices) to communicate with one another using Python scripts.

This project made use of several network programming techniques including: socket programming, content providers, total-causal ordering, key-value storage based on Chord, request routing, and quorom replication.

Operating System Design / Implementation

Designed, implemented, and debugged (using GDB) a BSD-like operating system through Virtual Box as part of a two-person team. Synchronization primatives (locks, condition variables, reader / writer locks) and general system calls (open, read, write, fork, _exit, etc.) are just a few of the aspects of OS development that were implemented through the lifetime of this project.
